Futhark

Futhark is my own engine written in C. I am making it as a passion project.

The name Futhark is just a code name the engine has no official name as of yet.

I came up with the name not only because I am Swedish, but it was actually originally written in the Odin programming language.

It has no connection to the Futhark programming language, I didn't even know it existed before making this website.

Libraries

Futhark has a few utility libraries. These are made to be easily replaceable or being put into a new project.

As such, most are built as a single-header library in the style of the stb libraries.

FHOS - Futhark OS

FHOS is a single-header cross-platform OS layer.

This is part of the platform layer used in the engine.

It implements allocators and a context system.

The context is a struct that is passed in as the first argument with many functions.

The context has a pointer to two allocators, one of which is meant for temporary memory.

FHFor - Futhark For Loops

FHFor is a header with some for loop macros.

FHDuck - Futhark Duck Typing

FHDuck is a header with macros for array functionality.

The two main "types" of FHDuck are arrays and lists.

  • Arrays have a fixed count (you can resize them, but this will realloc the data).

  • Lists have a dynamic count and a capacity. Only when the count exceeds the capacity, the array will realloc.
